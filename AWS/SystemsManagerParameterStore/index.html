SystemsManagerParameterStore

Systems Manager Parameter Store

이 서비스는 AWS에서 제공을 하는 비밀번호 및 구성 데이터 관리 서비스이다.
파라미터를 생성하여 파라미터의 유형과 값을 생성후 불러와 사용을 한다.

이곳

사용은 아직 해보지 않았으며, 보통 서버의 코드를 Github에 올리는데 민감한 데이터는 어디에 저장을 하는지 몰라서 찾아보던중 찾은 서비스이다. 이외에도 Secrets Manager라는 서비스가 있는데 이는 좀더 보안이 좋은 서비스인거 같으나 비용 문제로 Systems Manager Parameter Store서비스를 이용해 보기로 하였다.
